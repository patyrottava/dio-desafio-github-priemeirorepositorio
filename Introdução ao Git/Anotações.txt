Listagem de comandos Básicos

    >dir:listagem de pastas
    >cd
    >cls
    >ls
    >git config --global user.name: Setar usuário
    >git config --global user.email: Setar email
    >git config --global core.editor: Setar editor
    >git config --list: Listar configurações
    >git init: Criar novo repositório
    >git status: Verificar estado dos arquivos/diretórios
    >git add meu_arquivo.txt: Adicionar um arquivo em específico
    >git add .: Adicionar todos os arquivos/diretórios
    >git commit meu_arquivo.txt: Comitar um arquivo
    >git commit meu_arquivo.txt meu_outro_arquivo.txt: Comitar vários arquivos
    >git commit meuarquivo.txt -m "minha mensagem de commit": Comitar informando mensagem 
    >git rm meu_arquivo.txt: Remover arquivo
    >git status: fornece algumas informações sobre a branch
    >Git clone: baixar o código-fonte existente de um repositório remoto
    >Git push:  envia e salva suas confirmações no repositório remoto
    >git pull: obter atualizações do repositório remoto
